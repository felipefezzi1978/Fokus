# Fokus

Projeto exemplo do curso **JavaScript DOM** da Alura. Este repositório contém uma pequena aplicação/atividade chamada "Fokus" construída com HTML, CSS e recursos (imagens e sons). O foco é manipulação do DOM e interação no navegador.

**Status**: Em desenvolvimento

**Pré-requisitos**
- Navegador moderno (Chrome, Firefox, Edge, etc.)
- Opcional: `Python` ou `Node` para rodar um servidor local

**Como executar**

Opção 1 — Abrir localmente:

1. Abra o arquivo `index.html` no seu navegador (duplo clique ou abrir com o navegador).

Opção 2 — Abrir via terminal (PowerShell):

```powershell
start index.html
```

Opção 3 — Servidor HTTP simples (recomendado para recursos relativos):

Com Python:

```powershell
python -m http.server 8000
# Depois abra: http://localhost:8000
```

Com `npx serve` (Node):

```powershell
npx serve
# Depois abra a URL mostrada no terminal
```

**Estrutura do projeto**

- `index.html` — página principal
- `styles.css` — estilos
- `imagens/` — imagens usadas no projeto
- `sons/` — arquivos de áudio usados no projeto
